Creating and Hosting a Personal Site on GitHub

This Tutorial is simply named Creating and Hosting a Personal Site on GitHub. It is written by Jonathan McGlone, who is a experienced code ninja and seems like a likable guy. Only with this tutorial I managed to put up this site with its main features, so thanks a lot Jonathan!
